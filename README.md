## 🚀 Избранные проекты

### 🤖 [ESP32 Robot Car Control (Tauri)](https://github.com/GoooooPooooo/tauri-robot-control)
> Desktop-приложение на **Tauri + React + Rust** для управления роботом.  
> 🔹 Управление с ПК/смартфона через WiFi  
> 🔹 Поддержка прошивок: Arduino, Rust (esp-hal/esp-idf)  
> 🔹 HTTP API для реального времени

### 🦎 [Django Image Classifier](https://github.com/GoooooPooooo/django-webapp)
> Веб-приложение для классификации рептилий (**Варан / Хамелеон / Геккон**).  
> 🔹 Модель: **ResNet20 (ONNX)**, точность ~85%  
> 🔹 Backend: Django 4.2+, Frontend: HTML5/CSS3  
> 🔹 Поддержка S3/MinIO для хранения

### 🏭 [Remote Industrial Robot Control (FYP)](https://github.com/GoooooPooooo/FYP)
> Система удалённого управления промышленным роботом **MELFA RV-2AJ**.  
> 🔹 Двунаправленная связь через **MQTT + ESP32**  
> 🔹 Протокол RS-232 ↔ TTL ↔ UART  
> 🔹 Клиент: IoT MQTT Panel, MicroPython прошивка

### ♿ [Сурдобраслет на Arduino](https://github.com/GoooooPooooo/project_Surdobaslet)
> Социальный проект: носимое устройство для слабослышащих.  
> 🔹 Преобразование звука в вибрацию и световую индикацию  
> 🔹 Адаптивная логика: 3 уровня громкости → 3 цвета + интенсивность вибрации  
> 🔹 Бюджетное решение на базе Arduino Uno/Nano
